# Oqylyq Moodle Plugin (local_oqylyq)

## Description
This local plugin integrates Oqylyq/TrustExam features into Moodle: secure examination, AI-powered proctoring, identity verification, access control, and exam analytics.

It is designed for organizations that require reliable monitoring and integrity checks during online assessments.

## Installation
1. Copy this plugin into the directory:
2. (Optional) Run:
3. Log in as administrator and navigate to:
    **Site administration → Notifications**
    to complete the installation.

## Configuration
Navigate to:  
  **Site administration → Plugins → Local plugins → Oqylyq**

From there you can configure:
- API endpoint  
- Access token  
- Integration parameters
